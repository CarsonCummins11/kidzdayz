Tivoli

Thursday, December 19, 2019

Tivoli Gardens and Amusement Park is the second oldest amusement park in the world, opened in 1843. I was so grateful to be there at Christmas-time.  The decorations are truly amazing! We didn't ride the rides because a) it was freezing, b) I didn't want to get sick and, c) Lucy had already done them - thankfully :)  It was so cold but such a treat to be there, loved it!!
